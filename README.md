# Kauri

Plataforma full-stack de gestão de finanças pessoais desenvolvida com Next.js, Java Spring Boot e PostgreSQL.

O Kauri tem como objetivo ajudar usuários a organizarem suas finanças de forma simples, clara e visual, oferecendo um dashboard moderno para acompanhamento de receitas, despesas e categorias de gastos.

---

## Visão do Projeto

Kauri é um sistema de controle financeiro pessoal que permite registrar transações, categorizar receitas e despesas e visualizar a saúde financeira através de dashboards e gráficos.

O projeto está sendo desenvolvido como uma aplicação full-stack moderna com foco em:

- organização financeira
- visualização clara dos dados
- arquitetura escalável
- boa experiência do usuário
